## Project Overview

This repository contains our group's analysis of the Titanic passenger
dataset. The project has two goals: (1) build accurate visualizations that
clearly communicate real patterns in the data, and (2) build intentionally
misleading visualizations that demonstrate how design choices can distort a
reader's understanding.

## Dataset

We used the [Titanic dataset from Kaggle](https://www.kaggle.com/c/titanic/data),
which contains demographic and survival information for passengers aboard
the RMS Titanic (passenger class, sex, age, fare, survival outcome, etc.).
