# DATA6550 Visualization Ethics and Communication

## Project Overview

This repository contains the work completed for the **DATA6550 Visualization Ethics and Communication** group project. The goal of this project is to explore the Titanic dataset through ethical data visualization while also demonstrating how visualization choices can unintentionally or intentionally influence the interpretation of data.

The project includes exploratory data analysis, accurate visualizations, intentionally misleading visualizations, documentation of team collaboration, and a final report.

---

## Dataset

This project uses the **Titanic** dataset from Kaggle.

**Dataset:** https://www.kaggle.com/c/titanic/data

The dataset contains demographic and travel information for passengers aboard the RMS Titanic, including passenger class, age, sex, fare, family relationships, embarkation port, and survival status.

## Project Objectives

- Explore and understand the Titanic dataset.
- Create clear and accurate visualizations that communicate meaningful insights.
- Design intentionally misleading visualizations to demonstrate how presentation choices can affect interpretation.
- Discuss the ethical implications of misleading visualizations.
- Collaborate effectively through GitHub and shared documentation.
